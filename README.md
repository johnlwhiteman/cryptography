



# Utilities





## Web

* Install latest express at [express](https://expressjs.com/) ... version >= 4.
* Install Postman to make requests to our server at [postman](https://www.getpostman.com)

Create a web directory folder

Create an empty package.json file

```
$ mkdir web
$ cd ./web
$ npm init -y
$ npm install express --save
```
